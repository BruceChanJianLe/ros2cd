## Installation

For bash users

```bash
curl -s https://raw.githubusercontent.com/BruceChanJianLe/ros2cd/refs/heads/master/install.sh | bash
source ~/.bashrc
```

For zsh users
```bash
curl -s https://raw.githubusercontent.com/BruceChanJianLe/ros2cd/refs/heads/master/install.sh | bash
echo "source $HOME/.bash_aliases" >> $HOME/.zshrc
source $HOME/.zshrc
```
